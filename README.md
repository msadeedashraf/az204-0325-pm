### To setup the project
>git init
>git status

>git branch
>git branch -r

>git branch --set-upstream-to=origin/main main
or
>git branch --set-upstream-to=origin/master master

>git checkout -b main

>echo "# Initial commit" > README.md
>git add README.md
or 
>git add .
>git commit -m "Initial commit"

>git push --set-upstream origin main
